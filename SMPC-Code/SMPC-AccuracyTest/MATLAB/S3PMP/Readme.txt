该文件是安全三方矩阵乘法协议得使用说明。

【文件组织架构】
1.RandMatrixGen2.m用于2PMP内测精度实验，输出内容包含：原始数据矩阵A、随即混淆矩阵Ra、混淆后的数据矩阵A_hat；

2.RandMatrixGen3.m用于3PMP内测精度实验，输出内容包含：原始数据矩阵A、随即混淆矩阵Ra、混淆后的数据矩阵A_hat；

3.RBMG.m用于矩阵增广扩大后在2PMP和3PMP中的随机矩阵预生成阶段，按照分块思想实现输出大尺度的非满秩且阶数对齐的随机矩阵；

4.RBMG_Test.m用于RBMG.m随机生成函数的调试、修改、生成，按线性关系优化生成秩缺矩阵；

5.RMG_2PMP.m用于多方计算中对2PMP协议调用的随机矩阵预生成阶段，输出为阶数对齐的随机矩阵（不涉及非满秩约束）；

6.RMG_3PMP.m用于多方计算中对3PMP协议调用的随机矩阵预生成阶段，输出为阶数对齐的非满秩随机矩阵（右上角元素未优化版）；

7.RMG_3PMP_Cheat.m用于多方计算中对3PMP协议调用的随机矩阵预生成阶段，输出为阶数对齐的随机矩阵（不涉及非满秩约束）；

8.RMG_3PMP_Enhanced.m用于多方计算中对3PMP协议调用的随机矩阵预生成阶段，输出为阶数对齐的非满秩随机矩阵（右上角元素优化版）；

9.RMG_3PMPTest.m用于RMG_3PMP.m随机生成函数的调试、修改、生成，按右上角第四模块元素阶数门限进行阶数全对齐优化；

10.Secure_3PMP用于单次安全三方乘法协议精度算法的调试；

11.Single_3PMP用于单次安全三方乘法协议计算精度测试的实验文件，最终以txt格式将最大相对误差MRE，均值绝对百分比误差MAPE，范数相
对误差分位数Q存储其中；

12.Ultimate_3PMP用于多次安全三方乘法协议计算精度测试的实验文件，最终以Excel形式将最大相对误差MRE，均值绝对百分比误差MAPE，范数相
对误差分位数Q存储其中；

【固定研究问题】
1.精度误差跟首位元素的大小是否有关系；
2.精度误差跟阶数分布的最大最小or区间长度是否有关系；
3.精度误差跟尾数的长短是否有关系；
4.精度误差跟矩维度是否有关系；

【新增研究问题】
1.对于非满秩阶数对齐矩阵的生成方法，一共提出了三种形式，①以目标矩阵中任意两行之间阶数差的平方和最小的loss行为对象，进行比
  行等比例线性缩放，实现秩亏矩阵；②原始矩阵扩增行列，构建大方阵，将原来的阶数对齐非满秩问题，转化为大矩阵的阶数对齐非满秩
  问题，通过Ra矩阵任意中间行的比例缩放实现非满秩构建；③基于极大无关基向量组通过线性组合的方式构造非满秩指定阶数矩阵。

2.对于方法①存在的问题是，loss最小的时候也可能存在两行之间阶数差距较大，这个时候取阶数中间值不管如何取，依然会引入大阶数误
  差，而且是一整行上下两个元素的阶数误差；

3.对于方法②存在的问题是，由于矩阵的扩增，导致计算复杂度变高，计算成本变大，但是优点也很明显，既可以实现对任意输入矩阵的阶
  数和非满秩的约束，同时该方法还有个潜在优点就是自检验正确性的方法变得更加高级，通过四个对角矩阵的加减可以不暴露任何信息的
  前提下实现自检；

4.对于方法③存在的问题是，该方法通过一个非满秩的基向量的线性组合构建阶数对齐矩阵，虽然满足非满秩条件，但是由于最右上角元素
  无法直接实现阶数对齐，需要进行一个筛选，而经过试验得出结论是对于某些阶数分布的矩阵是不存在阶数对齐且非满秩矩阵的，因此该
  方法存在一定局限，优点是以牺牲一个元素的精度或者说引入一个元素带来的精度误差，相比方法①带来的误差较小；

【研究结论】
1.精度误差跟首位元素的大小无明显相关性。因为首位元素的大小无论为什么，最终混淆了随机矩阵之后的新矩阵都会导致其首位元素变化
  从而失去其固有特征；
 【修正】：这里出现了一个反常情况，对于3PMP出现了一种现象，精度和首位元素存在一定关系，首位元素越大，误差也越大。这里个人猜
  测，可能跟3PMP算法内部涉及到两个点有关，一个是B_hat分解过程一个是矩阵阶数对齐过程均涉及到矩阵的求逆，很大可能是从这两个地
  方引入了计算误差；

2.精度误差跟阶数分布的区间长度无明显关系，但是与阶数分布的最大最小值的落点有强相关性。


3.精度误差跟尾数长短无明显相关性。原始矩阵元素尾数在1~15位以内的情况下，并不影响混淆了随机矩阵RA后形成的A_hat矩阵的有效位数，
  因为A_hat矩阵的位数跟算法本身无关，跟计算存储精度double类型有关。

4.精度误差跟维度具有一定程度的相关性。当两方矩阵相乘时，维度分别为(m,n)(n,s)的矩阵，其涉及到的精度主要以单个元素为主，此时
  精度主要体现在n维度上，n决定了累加的次数；而当三方矩阵或者多方矩阵相乘时，维度分别为(m,n)(n,s)(s,t)的矩阵，其涉及到的精度体
  现在维度n和s上，第一次乘法涉及到n次累加，第二次乘法涉及到s次累加。因此为了简化讨论，我们先以方阵作为实验对象。
